# Google WebSearch

This is a PHP class that provides an easy interface to the Google Web Search JSON API documented here:
[http://code.google.com/apis/ajaxsearch/documentation/reference.html#_intro_fonje](http://code.google.com/apis/ajaxsearch/documentation/reference.html#_intro_fonje)

The Google API that we use is the same as is used in the Google AJAX Search API only this class cuts out the JavaScript. 

This class makes it simple to get customized Google Web Search results in a PHP page.
